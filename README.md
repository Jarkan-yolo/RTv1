# RT

./rt

usage: rtv1 [-s source] [-w width] [-h height]

-s : Set the specified source file

-w : Width of the window, default 1024px

-h : Height of the window, default 720px

--help : Show help

-a : Set the antialiasing technique to the image



Filters:

1 = Disabled

2 = Sepia

3 = Black and White

4 = Revert

Press ~
